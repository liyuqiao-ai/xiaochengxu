# 管理后台

农业零工数字化平台 - 管理后台系统

## 功能模块

1. **数据概览** - 平台数据统计和实时监控
2. **用户管理** - 用户审核、信用管理、角色管理
3. **订单监控** - 订单状态监控、异常处理
4. **财务管理** - 资金流水、对账管理、提现审核
5. **系统配置** - 费率设置、规则配置、权限管理

## 技术栈

- 微信小程序原生框架
- TypeScript
- 云函数后端

## 目录结构

```
admin-dashboard/
├── pages/
│   ├── login/          # 管理员登录
│   ├── index/          # 数据概览
│   ├── users/          # 用户管理
│   ├── orders/         # 订单监控
│   ├── finance/        # 财务管理
│   └── system/         # 系统配置
└── components/         # 公共组件
```

## 开发说明

管理后台作为独立的小程序模块，需要单独配置和部署。
