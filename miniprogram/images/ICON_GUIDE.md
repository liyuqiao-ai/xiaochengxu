# TabBar图标说明

## 需要的图标文件

由于无法直接创建图片文件，请手动添加以下图标：

1. **home.png** (64x64px)
   - 颜色：绿色 (#4CAF50)
   - 图标：首页/房子图标
   - 用于：未选中状态的首页tab

2. **home-active.png** (64x64px)
   - 颜色：深绿色 (#45a049)
   - 图标：首页/房子图标（高亮）
   - 用于：选中状态的首页tab

3. **profile.png** (64x64px)
   - 颜色：灰色 (#7A7E83)
   - 图标：用户/个人图标
   - 用于：未选中状态的我的tab

4. **profile-active.png** (64x64px)
   - 颜色：深绿色 (#4CAF50)
   - 图标：用户/个人图标（高亮）
   - 用于：选中状态的我的tab

## 获取图标的方式

1. 使用在线图标库：
   - iconfont (https://www.iconfont.cn/)
   - iconify (https://iconify.design/)
   - 搜索"home"和"user"图标

2. 使用设计工具：
   - Figma
   - Sketch
   - Adobe XD

3. 临时方案：
   - 可以使用纯色方块作为占位符
   - 或者使用微信开发者工具自带的图标

## 图标规范

- 尺寸：64x64px（128x128px @2x）
- 格式：PNG（支持透明）
- 颜色：符合品牌色（绿色系）
