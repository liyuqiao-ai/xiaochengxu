# 图标文件说明

此目录用于存放 tabBar 图标文件。

## 需要的图标文件

1. `home.png` - 首页图标（未选中状态）
2. `home-active.png` - 首页图标（选中状态）
3. `profile.png` - 我的图标（未选中状态）
4. `profile-active.png` - 我的图标（选中状态）

## 图标要求

- 尺寸：81px × 81px
- 格式：PNG
- 颜色：建议使用灰色（未选中）和绿色（选中）

## 临时解决方案

如果暂时没有图标文件，可以：
1. 使用在线图标生成工具创建
2. 使用微信开发者工具自带的图标
3. 暂时移除 tabBar 配置（如果不需要底部导航栏）
